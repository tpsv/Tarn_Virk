üìù 4. Data Cleaning & Preprocessing

üîπ Step 4: Text Cleaning

Phishing emails often contain special characters, links, and spam words. We’ll clean them using Regex and NLP.

import re
import nltk
from nltk.corpus import stopwords
from nltk.tokenize import word_tokenize

# Download required NLTK resources
nltk.download("stopwords")
nltk.download("punkt")

def preprocess_text(text):
    """Clean and preprocess email text."""
    text = re.sub(r'\W', ' ', text)  # Remove special characters
    text = text.lower()  # Convert to lowercase
    tokens = word_tokenize(text)  # Tokenization
    stop_words = set(stopwords.words("english"))
    text = ' '.join(word for word in tokens if word not in stop_words)  # Remove stopwords
    return text

# Apply preprocessing
emails["processed_text"] = emails["email_body"].apply(preprocess_text)

Some explanation of the above code :

‚úÖ Why do we do this?

  • Lowercasing → Ensures "FREE" and "free" are treated the same
  • Removing special characters → Cleans text noise
  • Stopword removal → Eliminates unnecessary words

üìä 5. Feature Engineering: Convert Text to Numerical Data

Since Machine Learning models don’t understand text, we need to convert it into numbers.

üîπ Step 5: Use TF-IDF Vectorization

TF-IDF (Term Frequency(How often the same terms appears in a document) - Inverse Document Frequency(Assigns a lower importance value to common words like the , is, are)) represents text in numerical form based on word importance.

from sklearn.feature_extraction.text import TfidfVectorizer

# Initialize TF-IDF Vectorizer with a limit of top 5000 features
vectorizer = TfidfVectorizer(max_features=5000)

# Transform processed text into a TF-IDF feature matrix

import tkinter as tk
from tkinter import messagebox
from model import predict_email  # Import the prediction function

# Function to handle button click
def check_email():
    email_text = email_entry.get("1.0", tk.END).strip()
    if not email_text:
        messagebox.showwarning("Warning", "Please enter an email text!")
        return
    prediction = predict_email(email_text)
    messagebox.showinfo("Result", f"Prediction: {prediction}")

# Create the main window
root = tk.Tk()
root.title("Phishing Email Detector")
root.geometry("400x300")
root.configure(bg="#f0f0f0")

# Title Label
title_label = tk.Label(root, text="Phishing Email Detector", font=("Arial", 14, "bold"), bg="#f0f0f0")
title_label.pack(pady=10)

# Email Input Box
email_entry = tk.Text(root, height=5, width=40, font=("Arial", 12))
email_entry.pack(pady=10)

# Check Button
check_button = tk.Button(root, text="Check Email", font=("Arial", 12), bg="#4CAF50", fg="white", command=check_email)
check_button.pack(pady=10)

# Run the Tkinter event loop
root.mainloop()
